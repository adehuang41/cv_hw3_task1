# Outputs

This is the only result aggregation area for Task 1.

## Conventions

- Put previews under `previews/`
- Put final renders under `renders/`
- Put evaluation artifacts under `eval/`
- Put checkpoints and model states under `checkpoints/`

## Checkpoint layout

- `checkpoints/object_A_2dgs/`
- `checkpoints/background_2dgs/`
- `checkpoints/object_B_threestudio/`
- `checkpoints/object_C_magic123/`

Checkpoint files, generated media, and large artifacts should not be committed to Git. Document share links in README files later when needed.
