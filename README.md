# CV Final Project Task 1

This repository contains the code, configurations, scripts, and report source for Task 1 of the computer vision final project.

## Project Layout

- `configs/`: experiment and pipeline configurations.
- `scripts/`: reconstruction, AIGC asset processing, Blender utilities, Gaussian fusion, and evaluation scripts.
- `pipelines/`: notes and reproducible workflow records for the four task branches.
- `report/cn_draft/main.tex`: Chinese LaTeX report source.
- `report/cn_draft/generated/` and `report/assets/`: report figures.
- `report/cn_draft/main.pdf`: compiled report.
- `data/`, `outputs/`, and model checkpoints are not stored in Git.

## Task Branches

1. Multi-view reconstruction for Object A and background D.
2. Text-to-3D generation for Object B.
3. Single-image-to-3D generation for Object C.
4. Mesh-to-Gaussian fusion and final rendering.
